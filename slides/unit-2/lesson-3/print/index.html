Introduction to Increment

In Java, there are ways to increase/decrease a variable without writing the whole equation!

1
2
3
4
5
6
7
x += 7; // The same as -> x = x + 7; 

y /= 3; // The same as -> y = y / 3 ; 

// Both are the same as x = x + 1
x++; // use then change
++x; // change then use
