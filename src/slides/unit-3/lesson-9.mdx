# Relational Operators and if/else


---

## if statements

```raw
if (test) { /// test = boolean or expression 
    <statement>;
    <statement>;
}
```

---

## if statements

```raw
if (test) {
    execute this code
    when test is true
}

execute this code always!
```

## if / else statements

```raw

    <statement>; 
    <statement>; 
}

<statement>;
```
