# Nested for Loops

---

## The Challenge:


Write code that counts to 1000 in 11 lines. 

**The Catch**...You cannot use a loop that executes more than 10 times.

---

## Single Loop

```java
for (int i = 1; i <= 3; i ++) {
    System.out.println("Hello");
}
```

---

## Nested Loop

```java
for (int i = 1; i <= 3; i ++) {
    for (int j = 1; j <= 3; j++) {
        System.out.println("Hello");
    }
}
```
